Phrase Hunter OOP Project

This was my most recent project using JavaScript, CSS, and HTML. I was the sole creator/developer for the project.

The directory pulls pictures and information from a third party website to compile a fake emplyee directory.

Every time the page is refreshed a new panel of employees are generated into twelve card slots.

Users have the ability to interact with all of the cards shown and clicking on them provides an expanded view with more employee information.

I worked on this system for a TreeHouse project.

Technologies:
  • - Python
  • - Django + Django REST Framework
  • - React JS
  • - Postgres
  • - AWS (S3 & RDS)
  • - Zingcharts - Trending & graphing test results
  • - Google Maps API
  • - XML2PDF - PDF generator
