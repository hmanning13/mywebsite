Phrase Hunter OOP Project

This is an object oriented programming project where I was once again the sole creator and developer.

The game is basically a themed verison of hangman with a few random words and phrases chosen by myself for the user to guess.

Every player has five lives to guess the phrase/word correctly.

The application features working buttons and an up to date lives panel so the player can check their progress.

When refreshed/restarted the lives reset to full and another random phrase/word is chosen again.

Technologies:
  • - JavaScript
  • - HTML
  • - CSS
  • - OOP (Object-Oriented Programming)
