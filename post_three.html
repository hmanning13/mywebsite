Interactive Form

This is an JavaScript project where I was in charge of creating a woking and interactive sign-up form.

The interactive form has four different sections.

Every user has to fill out their name & email into the text field which will only accept valid responses.

The page then has a t-shirt section where the user must fill out their preference of size and design.

Lastly, the page features a "Register for Activities" page that will not let users select conflicting events. After the register section, there is a payment section to wrap the whole project up. In the payment section, users can select different payment types and only valid entries into the text field will be accepted.

Technologies:

  • - HTML
  • - CSS
  • - Interactive Forms
  • - Form Validation
